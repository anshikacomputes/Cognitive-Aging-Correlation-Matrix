# Cognitive-Aging-Correlation-Matrix
A Python-based visualization tool to analyze correlations between aging and cognitive decline in clinical datasets.

**Author:** Anshika Mishra 

## Project Overview
This project simulates and visualizes the relationship between biological aging and cognitive performance metrics (Memory, Focus, and Language). Understanding these correlations is fundamental to research in neurodegenerative diseases like Alzheimer's and Dementia.

## Technical Implementation
* **Data Processing:** Utilized `Pandas` to structure simulated clinical data.
* **Statistical Analysis:** Calculated Pearson correlation coefficients to identify trends across age groups.
* **Visualization:** Implemented `Seaborn` heatmaps to create an intuitive, color-coded correlation matrix (R-values).

## Research Context
Inspired by large-scale aging studies (e.g., SANSCOG), this tool demonstrates the ability to handle multidimensional biological data and extract meaningful statistical insights. 

## Libraries Used
* Pandas
* Seaborn
* Matplotlib
